Name: Peak Time  
Role: Discrete Dimension  
Type: Calculated Field  
Formula:

```
IF DATEPART('hour', [Interval]) >= [Peak Period Start] AND
DATEPART('hour', [Interval]) < ([Peak Period Start] + [Peak Period Length (Hours)]) THEN 'Peak Time'
ELSEIF
([Peak Period Length (Hours)] + [Peak Period Start]) > 23 AND
DATEPART('hour', [Interval]) < ([Peak Period Start] + [Peak Period Length (Hours)] - 24) THEN 'Peak Time'
ELSE 'Off-Peak Time'
END
```
